
# Rod Building Hardware Contoller UI


## Description:
###### UI to control my fishing rod dryers speed, fan speed, enclosure temperature and lights remotely.
###### Live demo version: https://hardwareui.herokuapp.com/ 


## Technologies Used:
#### Software
  1. Express.js
  2. Angular.js
  3. Angular Route
  4. Node.js
  5. Socket.io
  6. Johnny-Five
  7. Body-Parser
  8. PG - version
  9. GIT / GitHub

#### Hardware
  1. Arduino ONU



## Install App:
  1. Download zip file
  2. Open terminal and navigate into folder
  3. Run commands:
    - ``` npm install ``` Installs Node Packages including dependencies from package.json
    - ``` npm start ``` Starts server
    - ``` npm test ``` Starts server using Nodemon
  4. Server is listening on port: 5000
